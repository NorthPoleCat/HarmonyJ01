名[人]は　名[人]に　[句简体]と　言いました

向第三者转述他人所说的话。

首次出现于初级上，24课

备注：如果转述某人反复说的事情，使用"〜と　言っています"

例句：

陳ちんさんは　パーティーに　行いく　と　言いいました。

老陈说要去参加联欢会。

森もりさんは　中ちゅう国ごく語ごの　試し験けんは　難むずかしかった　と　言いいました。

森先生说汉语考试很难。

小野おのさんは　李りさんに　ちょっと　休やすみたい　と　言いいました。

小野女士跟小李说想休息一下。

来らい週しゅうの　ハイキングに　張ちょうさんも　行いきたい　と　言いって　います。

小张说他也想参加下星期的郊游。
